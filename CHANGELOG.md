# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-09

### Added
- Initial release of RscScan
- Multi-threaded vulnerability scanner for Next.js Server Actions (CVE-2025-55182)
- Electron desktop application with cross-platform support (Windows, macOS, Linux)
- Multi-language support (English, Persian, Russian, German, Chinese)
- Real-time progress tracking and statistics dashboard
- Export functionality (JSON, CSV)
- Dark and Light theme support
- In-app toast notifications
